# SaneUI Security

SaneUI is a shared SwiftUI component package for SaneApps. It does not collect customer data by itself.

## Scope

Useful reports include:

- A shared component accidentally sending data over the network
- Sensitive diagnostics copied into public bug-report text by default
- License or keychain UI exposing secrets in logs or screenshots
- Components that encourage unsafe permission or privacy copy in downstream apps

## Reporting

Email security or privacy concerns to hi@saneapps.com.

Do not include private customer logs, screenshots, license keys, or personal data in a public GitHub issue.
